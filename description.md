Página de inscrição para a pós-graduação no Programa de Pós-Graduação em Modelagem e Métodos Quantitativos (PPGMMQ) da Universidade Federal do Ceará (UFC).

## Tecnologias

- HTML
- CSS
- JavaScript
- Bootstrap (v5.3)

## Estrutura do projeto

- index.html: Página principal
- style.css: Estilos CSS
- script.js: Script JavaScript
- Bootstrap: Biblioteca de componentes do Bootstrap v5.3 via CDN

## Features

O candidato deverá fazer o upload de muitos documentos (comprovantes) para cada seção do formulário. Então a página deverá ter campos para upload com um botão para coleta do caminho do arquivo e a sua descrição.
